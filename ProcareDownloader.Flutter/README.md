# Procare Photo Downloader (Flutter)

Flutter implementation for the Procare Photo Downloader app.

## Build and run

From `ProcareDownloader/ProcareDownloader.Flutter`:

```powershell
flutter pub get
flutter run
```

Useful follow-ups:

- `flutter test`
- `flutter build apk` (or `flutter build ios` on macOS)
- `flutter run --release`
